Вебинар "Что может фронтендер"

Цель этого вебинара: познакомить новичков с работой фронтендера-верстальщика и завлечь их на продолжение обучения:) В отличие от курса, где все объясняется с нуля, тут сразу демонстрируются разные интересные штуки.

Студенты увидят как происходит верстка с макета в Фигме, познакомятся с языками HTML, СSS и немного JavaScript, увидят, как сайт выкладывается на хостинг Github Pages, смогут оценить процесс работы и решить, насколько им это интересно.

Примерная длительность: 3,5 часа с ответами и перерывом. Можно разбить на две части.

Провел этот вибинар в ИПАП

План вебинара:

  1. Что такое фронтенд и что можно в нем делать
  2. Работа с макетом
  3. HTML
  4. CSS
  5. Раскладки
  6. Анимации и переходы
  7. JS
  8. Выкладываем сайт на Github Pages
