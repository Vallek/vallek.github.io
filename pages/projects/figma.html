Вебинар "Фигма для верстальщика"

Задача вебинара - познакомить студентов с одним из популярных современных онлайн редакторов для верстальщика, которые приходят на смену Фотошопу.

Этот вебинар рассчитан на самых новичков, которые только начинают изучать верстку.

В начале вебинара я рассказываю про состояние рынка редакторов для верстальщика, их место в рабочих процессах, плюсы и минусы онлайн редакторов по сравнению с фотошопом. Основная часть вебинара - разбор самых базовых задач при верстке с макета в фигме.

А именно:

  • Размеры
  • Расстояния
  • Цвета
  • Шрифты
  • Содержимое

На вебинаре показана работа с разными макетами, включая мой собственный и встроенные в фигму. Но основным может стать макет из вашего учебного заведения.

В конце, как обычно, ответы на вопросы.
