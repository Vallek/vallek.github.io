Лекция "CSS Анимации"

На этом вебинаре вы познакомитесь с основами создания анимаций с помощью CSS.

После небольшого введения в анимацию, вы научитесь использовать CSS Переходы и CSS Анимации на реальных примерах. А также узнаете, где и как искать источники вдохновения для создания своих анимаций.

План занятия

  1. Презентация "Введение в веб-анимацию"
  2. Анимируемые свойства
  3. CSS Transitions
  4. Пример (анимация декоративных псевдоэлементов)
  5. CSS Анимации
  6. Примеры (анимация тряски иконки и исчезновение/появление попапа)
  7. Где искать примеры и источники вдохновения
