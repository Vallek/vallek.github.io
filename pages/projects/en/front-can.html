Webinar "What a Frontend Developer Can Do"

The goal of this webinar is to introduce newcomers to fronend developer work and to attract them to continue the course. As opposed to the full course where everything is explained from ground up, this webinar is jumping right into the interesting stuff - practice.

Students will get familiar with figma workflow, HTML, CSS (including animations) and a little bit of JS. They will learn how to use Github Pages as hosting and will be able to consider if they would like working as frontend developer themselves.

Duration: about 3,5 hours with answers and a breaks. Could be divided into two parts.

Held the webinar in IPAP

Plan:

  1. What is frontend
  2. Working with design template
  3. HTML
  4. CSS
  5. Layouts
  6. Animations and transitions
  7. JS
  8. Github Pages
