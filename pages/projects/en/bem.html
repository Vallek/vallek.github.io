Introduction to BEM Methodology

In this class, you will get familiar with the BEM methodology. It solves many potential frontend problems, helps to organize the code and file structure of the project.

You will learn the different approaches to class naming, the basic principles of the BEM methodology, see them in action on a live site and learn how to use them.

Plan

  1. Approaches to naming classes
  2. Possible problems
  3. BEM
  4. Site header on BEM
  5. Non-obvious problems
  6. Examples in the wild
  7. bemgo tool
