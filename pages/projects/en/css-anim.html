CSS Animations

Learn the basics of creating animations using CSS.

After a little introduction to animation, you will learn how to use CSS Transitions and CSS Animations on real examples. And also learn where and how to look for sources of inspiration to create your own animations.

Plan

  1. Introduction to web animation presentation
  2. Animatable CSS properties
  3. CSS Transitions
  4. Example (animation of decorative pseudo-elements)
  5. CSS Animations
  6. Examples (shaking icon popup appear/disappear)
  7. Sources of examples and inspiration
